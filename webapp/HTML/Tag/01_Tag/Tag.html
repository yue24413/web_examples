标签的基础学习

标签部分基本知识

一.语法

1.标签成对出现,中间包裹内容。

2.尖括号<>里放英文字母,即标签名。

3.结束标签比开始标签多／。例如:加粗标签 strong,例:<strong></strong>

4.文件后缀名是: .html

5.双标签:成对出现的标签;
单标签:只在开始出现,无结束。

例如单标签:
换行标签:<br>水平线标签:<hr>
扩展:
空格字符:&nbsp;
转义字符:左<&lt;  右>&gt

二.htm1的基本骨架

1.快捷键: !（英文）+ Enter/Tab

 !输入在.htm1文件上之后,再输入Enter/Tab。

2.html:整个网页。

3.head:网页头部,存放给浏览器看的代码,如 css。

4.body:网页主体,存放给用户看的代码,例如:图片,文字等。

5.title:网页的标题。

6.具体框架展示:

<htm1>
 <head>
   <title>网页标题</title>
 </head>
 <body>
  (网页主体)
 </body>
</html>

三.标签的关系

作用:明确代码的书写位置。

1.父子关系（嵌套）。

2.兄弟关系（平级）。

Tab 和 shift Tab 缩进

五．常见的标签

1.标题标签

一般用在新闻标题,文章标题...
标签名:h1~h6
h1标签在一个网页中只能用一次,用来放新闻标题或者网页logo。

2.段落标签

标签名:p(双标签)
显示特点：独占一行(换行),段落间有空隙。

3.换行与水平标签

(1).换行标签:<br>
浏览器不识别Enter键换行,所以文本需要换行必须有标签。(有需要就加)
(2).水平线标签:<hr>
这两个都是单标签。

4.文本格式化标签

常见的有:加粗,倾斜,下划线,删除线
<strong> / <b>    加粗标签
<em> / <i>   倾斜标签
<ins> / <u>   下划线标签
<del> / <s>   删除标签
尽量用左边的

5.图像标签

<img src = "图片的URL">
src用于指定图像的位置和名称,是<img>的必要属性。
图像的标签属性
   alt   替换文本  图片无法显示的时候显示文字
   title   提示文本   鼠标悬浮停在图片上面的时候显示文字
  属性名:"属性值"
  属性写在尖括号里面,标签名的后面,标签名和属性之间用空格隔开,不区分先后。
  <img src=" ./cat.jpg" alt="替换文本" title="提示文本">
