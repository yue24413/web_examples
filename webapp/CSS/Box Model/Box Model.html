盒子模型

一.内容区域、内边距、边框线、外边距

1.内容区域：width and height

2.内边距:padding

3.边框线：border；属性值：边框线的粗细，线条样式(solid / dashed)，颜色(无顺序)

4.外边距：margin;不会撑大盒子

注：
都有方位名词：top / right / bottom / left
多值写法：(巧记：顺时针)
top->right->bottom->left
top->right/left->bottom
top/bottom->right/left

二.尺寸计算

1.盒子的尺寸=内容尺寸+边框线(border)尺寸+内边距(padding)尺寸

2.给盒子加border/padding会撑大盒子

解决方式：手动减去/内减模式(box-sizing: border-box;)

三.版心居中

1.margin: 0 auto;

2.要求：盒子必须有宽度(或者继承了父级的宽度)，否则无法做减法

四.清除默认模式

1.解决方式：
*{
margin:0;
padding:0;
/*box-sizing: border-box;*/
}

2.去除无序列表前的小黑点符号：list-style:none;

3.在设置默认方式时，顺便把内减模式(box-sizing: border-box;)加上更好

五.元素溢出

1.属性名:overflow

2.属性值:hidden / scroll / auto

hidden:元素隐藏

scroll:溢出滚动(无论是否溢出，下方和右方都有滚动条位置)

auto:溢出滚动

六.外边距合并现象

1. 场景：垂直排列的兄弟关系，上下margin会合并

2.现象：两个较大值生效

七.外边距塌陷问题

1. 场景:父子级标签,子级添加上外边距margin会产生塌陷问题

2.现象：导致父级一起向下运动

3.解决方式：
(1).取消父级的margin,父级加padding
(2).父级设置overflow:hidden; 溢出隐藏，识别出父级正确的边缘，边缘外溢出隐藏
(3).父级设置border:top; 与(2)同理，识别出正确的父级边缘

八.行内元素内外边距问题

1. 场景:行内元素添加margin / padding,不生效垂直方向，只生效水平方向

2.解决方式:line-height来增加内部间距

九.圆角

1.属性名:border-radius

2.属性值：数字px / 百分比 (数字为圆角半径)

3.值可以写多值，方位名词与边距同理

4.常用类型：
(1).正圆(头像等):盒子本身就是正方形,宽高的一半 / 50% (超过50%还是正圆)
(2).胶囊(按钮、搜索框等):盒子本身就是长方形,高的一半

十.盒子的阴影

1.属性名:border-shadow

2.属性值:x轴偏移量 y轴偏移量 模糊半径 扩散半径 颜色 内/外阴影

3.x和y的偏移量必须要有,内/外阴影中默认是外阴影(内阴影用inset,不常用)

例: box-shodow: 2px 5px 10px 1px rgba(0,0,0,0.5)
