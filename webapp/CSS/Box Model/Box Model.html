三.边框线

1.border 属性值：边框线的粗细，线条样式，颜色(无顺序)

2.边框线粗细：solid / dashed

3.border方位名词：top / right / bottom / left

4.多值写法：(巧记：顺时针)
top->right->bottom->left
top->right/left->bottom
top/bottom->right/left

四.外边距

1.margin 方位名词与边框线同理

2.不会撑大盒子
