Float 浮动

1.块级标签在一行显示

2.属性名: float

3.属性值: left / right

4.具有顶对齐属性,具备line-bloak 属性

5.清除浮动问题:

(1).场景：浮动元素会脱标，若父级为设置高度，子级无法撑开父级的高度(页面可能存在错乱)

(2).解决：清除浮动

(3).解决方式:

- 额外标签法:在父元素内容最后添加一个块级元素,设置css属性: clear:both;

- 单伪元素法：
.clearbox::after{
content:"";
display:bloak;
clear:both;
}
- 双伪元素法：
.clearbox::before,.clearbox::after{    /* before为了防止外边界塌陷*/
content:"";
display:table;
}
.clearbox::after{
clear:both;
}
- overflow:

父元素添加css属性 overflow: hidden;

同理，检查父元素边界范围

会让浏览器检查元素边界，使浮动的子级可以撑开父级
