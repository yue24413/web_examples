Flex 布局

1.Flex模型不会产生浮动布局中的脱标现象,布局网页更灵活

2.Flex的组成

(1).给父元素设置 display：flex; 子元素可以自动挤压或拉伸(即使子元素设置了宽高,也会默认在父元素的一行里)

(2).组成成分：

弹性容器

弹性盒子

主轴：默认在水平方向

侧轴/交叉轴：默认在垂直方向

3.布局

(1)

  • 描述
  • 创建Flex容器
  • 主轴对齐方式
  • 侧轴对齐方式
  • 某个弹性盒子侧轴对齐方式
  • 修改主轴方向
  • 弹性伸缩比
  • 弹性盒子换行
  • 行对齐方式
  • 属性名
  • display:flex;
  • justify:content;
  • align-items;
  • align-self;
  • flex-direction;
  • flex
  • flex-wrap
  • align-content
  • 属性值
  •  
  • center   space-between  space-around  space-evenly  flex-start  flex-end
  • stretch(拉伸至铺满,不能设置尺寸)  center  flex-start  flex-end
  • (上同)
  • column  垂直方向,从上向下
  • 整数数字,表示占用父级尺寸的份数
  • wrap   nowrap
  • (和主轴对齐方式的属性相同)

(2).弹性伸缩比中,主轴方向尺寸靠内容撑开,侧轴默认拉伸

(3).行对齐方式对单行不生效(即未使用flex-wrap),flex-wrap放到要用子级的父级中
