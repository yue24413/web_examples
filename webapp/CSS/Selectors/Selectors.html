四种基本选择器

一.CSS三种引入方式

1.内部样式表

在head里的title双标签下开一处style双标签，里面用来书写内部样式。

2.外部样式表

CSS代码写在单独的.CSS文件中，在HTML中用link标签引入。

例如：<link rel="style sheet" herf'"./my.css">

3.行内样式

直接在标签里加style。例如：

<div style="color:red;font_size:30px;"<这是div标签</div>

二.标签选择器

使用标签名作为选择器，选中的标签设置相同的样式，在内部样式表中用。

三.类选择器

1.定义：
<style> .red{
color:red;
}
</style>

2.使用：
<p class=".red">我是红色的</p>

*.一个标签可以加多个类名，.类名有空格隔开。
*.要选有标志性的类名。

四.id选择器

定义：#id名
使用：标签 id="id名"
规则：同一个id选择器在一个页面中只能用一次。
例如：
<style>
#red{
color:red;
}
 </style>
</head>
<body>
<div id="red">div标签</div>
</body>

五.通配符选择器

1.作用：查找页面上所有标签，设置相同的样式。

2.通配符选择器：*，不需要调用选择器，浏览器会自动查找页面中所有标签，设置相同的样式。
