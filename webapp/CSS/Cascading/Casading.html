一.CSS三大特性

1.继承性

子级会继承父级的文字控制属性

注：当自己有样式，会用自己的样式

2.层叠性

相同的属性回覆盖，不同的属性会叠加

几个规则来自同一个源，且具有相同的元素选择器，有相同的优先级，顺序在最后的生效。

3.优先级

当一个标签用了多种选择器，基于不同种类的选择器的匹配规则。

公式：通配符< 标签 < 类 < id < 行内样式 < !important(慎用)

权重叠加计算公式：(行内样式，id选择器个数，类选择器个数，标签选择器个数)

注：从左到右依次比较个数，同一级个数多的优先级高，个数相同往后比较

！important例外，优先级最高，一般也不随便用

继承权重最低
